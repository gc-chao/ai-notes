
# 说明

## 简介

这是我个人在学习AI知识过程中整理的笔记。笔记内容分成六个模块，分别是传统AI算法、应用数学基础、传统机器学习、深度学习、应用专题以及代码示例。目前，已经提交的是应用数学知识、机器学习和代码示例三个模块，其余部分会陆续提交。

诚然，这份笔记不可能面面俱到，更多是作为一份速查手册或学习辅助。如果您需要深入了解某个概念的严格定义或数学推导，建议直接阅读经典教材或原始论文。另外，有些内容在语言上并未追求绝对的严谨性，甚至用了比较通俗的表达，目的是帮助理解。因此如果需要更精确的表述，还请以专业文献为准。

原则上，一篇文档可以无限制的优化下去，所以我遵从的是先从无到有，再从有到优的思路。可以说这份笔记并不完美，但适时公开能让它先发挥一些作用，再逐步完善。如果您发现了任何不严谨或错误的地方，欢迎通过Issue指出，我会及时修正。如果这份笔记对您有帮助，欢迎给它一个⭐。

另外，如果您对尚未发布的章节感兴趣，也可以私信联系我，我可以先发送目前的完整版本（虽然还需要打磨，但内容已经整理完毕）。

## 内容

这份笔记按主题分为六个独立文件，具体的内容安排如下。

- 经典AI算法（AI.md）：整理了传统AI领域的一些基础知识，包括知识表示、确定性推理、不确定性推理等内容。
- 应用数学知识（MATH.md）：汇总了机器学习中常用的数学知识，例如线性代数、概率论等。
- 机器学习（ML.md）：涵盖了机器学习的一些基本概念和传统模型。
- 深度学习（DL.md）：包含神经网络和深度学习的相关知识，以及训练和优化方法。
- 应用专题（APP.md）：记录了机器学习在一些领域的应用思路，例如自然语言处理等。
- 代码示例（CODE.md）：提供了一些典型算法的Python实现，以及训练代码的脚手架。

目前，应用数学知识（MATH.md）、机器学习（ML.md）和代码示例（CODE.md）这三部分已经提交，其余的部分正在打磨，后续会更新。

## 如何使用

### 在线浏览

由于GitHub默认不会渲染行内公式，因此在线阅读的体验可能不太好。如果只是快速浏览，可以直接在GitHub上看，但我更推荐下面的方式。

### 下载发布的PDF版本（推荐）

如果您只想阅读，不打算修改，则可以直接下载发布好的PDF版本。发布版本已经渲染了所有公式，排版也更适合阅读。

### 作为速查手册

如果您已有一定基础，需要快速回顾某个概念或算法，可以直接搜索关键词，或者根据目录快速定位到相关章节。笔记中的公式和关键结论可以帮助快速回忆。

### 自定义修改

如果您希望在这个笔记的基础上进行修改和扩充，或者用它来构建自己的讲义，您可以：

- 本地编辑：将仓库clone到本地，使用Markdown编辑器打开并进行修改（我用的是[Taio](https://taio.app/cn/)，它可以呈现与我笔记中一样的公式效果），然后导出成PDF或其它格式。
- 贡献与反馈：如果您发现了错误，或者想补充内容，欢迎提交Issue或Pull Request。

特别是，如果您需要快速生成一份针对特定主题的讲义，可以先复制相关章节的内容，再根据需要进行删减或重组。这样就能形成一份定制化的教学材料，而不必从头开始整理。

}
.markdown-body h2:before{
  counter-increment: level2;
  content: counter(level1) "." counter(level2) " ";
}
